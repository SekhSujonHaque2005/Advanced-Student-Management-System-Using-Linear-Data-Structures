# Advanced Student Record Management System Using Linear Data Structures

> A console-based Student Record Management System developed in **C++** to demonstrate the implementation of linear data structures and algorithms without using STL containers.

**Language:** C++

## About the Project

The Advanced Student Record Management System is a menu-driven console application designed to manage student records while implementing the fundamental concepts of data structures and algorithms.

Unlike traditional CRUD applications, this project focuses on implementing linear data structures from scratch for educational purposes.

The project demonstrates:

- Dynamic Memory Allocation
- Pointer Manipulation
- Linked Lists
- Stack
- Queue
- Searching Algorithms
- Sorting Algorithms
- File Handling

## Project Objectives

- Learn linear data structures practically.
- Implement linked lists without STL.
- Understand dynamic memory allocation.
- Practice searching and sorting algorithms.
- Build a real-world DSA application.

## How to Run

### Clone the Repository

```bash
git clone https://github.com/your-username/Student-Management-System.git
```

### Open the Project

Open the project in **Visual Studio Code**.

### Compile

```bash
g++ src/main.cpp -o student
```

### Run

```bash
./student
```

For Windows (MinGW):

```bash
student.exe
```
